# FARAS Dashboard – Decentralized Ride-Hailing Concept

A frontend dashboard prototype inspired by the FARAS lightpaper, exploring how blockchain can improve fairness and transparency in ride-hailing systems.

---

## Overview

This project simulates a ride-hailing platform where drivers keep more of their earnings compared to traditional platforms.

It is based on real-world experience in the taxi industry and explores how Web3 principles can be applied to create a more fair system.

---

## Features

- Add new rides dynamically
- Automatic calculation of driver earnings
- Comparison between FARAS (5%) and traditional platforms (25%)
- Driver loss calculation
- Driver savings percentage
- Average fare calculation
- Reset dashboard data
- Load sample data
- Data persistence using localStorage

---

## FARAS Concept

FARAS is a decentralized ride-hailing idea focused on:

- Lower platform fees
- Transparent payment systems
- Fair earnings for drivers
- Real-world application of blockchain technology

This dashboard is a simplified prototype showing how such a system could work.
