# DeFi DAO Portal – Agile Team Project

## Overview
This project is a Web3-inspired frontend application developed as part of an agile team assignment. The application simulates a decentralized platform where users can interact with staking pools, governance voting, and DAO proposals.

The main focus of the project was not only development, but also working in a team using agile methods such as Scrum.

---

## Features
- View platform statistics (Total Value Locked, Active Users, Proposals)
- Participate in simulated DAO governance voting
- Visualize voting results with progress bars
- Explore staking pools and token rewards
- Responsive layout for mobile, tablet, and desktop

---

## Technologies Used
- HTML5 (semantic structure)
- CSS3 (Flexbox, Grid, responsive design)
- Vanilla JavaScript (basic interactivity)
- Git & GitHub (version control and collaboration)
