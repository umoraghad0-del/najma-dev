# To-Do List App – Team JavaScript Project

## Overview
This project is a web-based to-do list application built with HTML, CSS, and JavaScript as part of a group assignment. The goal was to create a dynamic application where the interface is generated and managed through JavaScript instead of hardcoded HTML.

The app allows users to add, manage, filter, and store tasks in a practical and interactive way.

---

## Features
- Add tasks with optional deadline
- Mark tasks as completed
- Delete individual tasks
- Filter tasks by All / Active / Completed
- Search tasks
- Show remaining task count
- Clear all completed tasks
- Save tasks in `localStorage`
- Highlight overdue and current deadlines

---

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- localStorage
- Basic testing with Node.js

---

## Project Structure
- `index.html` – application structure
- `style.css` – styling
- `app.js` – main logic and event handling
- `dom.js` – dynamic HTML creation
- `storage.js` – localStorage functionality
- `test.js` – test for storage behavior
